# Inventory Management System

## Overview

This project is a lightweight inventory management system designed to track products, quantities, and inventory operations in a structured and efficient way. The application was built to strengthen hands-on experience with software development, database management, troubleshooting, and real-world business logic.

The system focuses on inventory organization, data accuracy, and operational efficiency while demonstrating practical programming and problem-solving skills.

---

## Features

- Add new inventory items
- Update existing inventory records
- Delete inventory entries
- Track product quantities
- Search and filter inventory data
- Organized inventory management workflow
- User-friendly interface and navigation
- Persistent data storage
- Error handling and input validation

## Installation

Clone the repository:

```bash
git clone https://github.com/smatneytech82/inventory-management-system.git
```

Navigate to the project directory:

```bash
cd inventory-management-system
```

Open the project in your preferred IDE and run the application.

## Usage

1. Launch the application
2. Add inventory items
3. Update quantities and product information
4. Remove outdated inventory entries
5. Track and organize inventory records
